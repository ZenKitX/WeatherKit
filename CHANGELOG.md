## 0.1.0

* Initial release
* WeatherService for fetching weather data
* Support for current weather, hourly and daily forecasts
* Built-in cache mechanism with SharedPreferences
* Result<T> type for error handling
* 6 error types (network, parsing, api key, rate limit, location not found, unknown)
* Support for fetching by city name and coordinates
* Comprehensive data models for weather information
